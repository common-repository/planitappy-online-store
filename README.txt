== Description ==

Take payment for memberships (automatic and once-off payments), physical products or other services straight from your WordPress website using the PlanItAppy Online Store plugin.

= What this plugin does =

This plugin displays a Booking Form on your website using a provided shortcode. You input your PlanItAppy store URL, and it displays the stores Booking Form using an iFrame. This involved calling Javascript from an external server. Any end user who wishes to make a booking will have to either register an account, or enter a username and password for an existing account on your PlanItAppy web store.

== Installation ==

1. Upload the plugin ZIP file to the `/wp-content/plugins/` directory
2. Unzip the plugin ZIP file.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Read the instructions under the plugins settings.
